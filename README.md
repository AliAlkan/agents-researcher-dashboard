# AgentsResearcher Dashboard

A dark-themed AI agent dashboard built from a Figma design using React, Tailwind CSS v4, and Vite.

## Features

- Collapsible left sidebar with navigation (New Chat, Chats, Quick Agents, Agents, Knowledge, Personas, Lists, Plans, Data Sources)
- Recent chat history in the sidebar
- Top navigation bar with EVOLVE logo, Dashboard/App Store links, search input, and user avatar
- Main content area with "Hand this off to an agent" hero section
- EvoResearcher input card
- Recents table listing agent runs with name, agent, created date, and status

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS v4
- Vite
- Radix UI components
- Lucide React icons

## Getting Started

```bash
# Install dependencies
npm install
# or
pnpm install

# Start development server
npm run dev

# Build for production
npm run build
```
